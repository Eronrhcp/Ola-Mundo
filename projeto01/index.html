Markdown

Neste parágrafo vai minhas anotações pessoais sobre Markdown.

Configurações

Negrito - Se escreve entre underline duplo __Markdown__ ou entre asteriscos duplos **Markdown**.

Itálico - Se escreve entre underline simples _Markdown_ ou entre asteriscos simples *Markdown*.

Riscado - Se escreve entre 'tios' duplo ~~Markdown~~.
